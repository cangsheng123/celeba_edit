# CelebA 人脸编辑系统（原型）

这是一个面向 **CelebA 人脸图像** 的轻量人脸编辑系统，提供精美 Web 界面与常见修图能力：

- 头发颜色调整
- 嘴巴大小调整
- 眼睛大小调整
- 瘦脸强度调整

> 技术路线：MediaPipe（人脸关键点）+ OpenCV（局部形变与颜色映射）+ Gradio（交互界面）。

## 快速启动

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

浏览器访问：`http://localhost:7860`

## 系统结构

### 1. UI 交互层（`app.py`）
- 上传图片并设置编辑参数。
- 一键触发编辑，右侧实时显示结果图。
- 提供系统说明折叠面板与“清空”按钮，便于演示。

### 2. 人脸感知层（`face_editor.py`）
- 使用 `MediaPipe Face Mesh` 提取人脸关键点。
- 若未检测到人脸，返回中文错误状态。

### 3. 编辑处理层（`face_editor.py`）
- 发色：基于头发掩码做 HSV 色相/饱和度映射。
- 嘴巴：对嘴部关键点区域执行局部缩放形变。
- 眼睛：对左右眼区域分别执行局部缩放形变。
- 瘦脸：对双颊区域施加高斯拉伸实现收脸效果。
- 参数安全：对所有输入参数做范围钳制，避免异常输入。

### 4. 输出反馈层
- 输出编辑结果图。
- 输出详细中文状态（包括本次实际参数），方便调参与追踪。

## 项目结构

- `app.py`: Gradio 前端界面与交互逻辑
- `face_editor.py`: 人脸关键点检测与编辑算法
- `requirements.txt`: 依赖列表
